# Parse-React-Login-Signup-Sample
Parse login and signup sample with React written in ES6.

#Built with
* react
* flux
* react-router
* parse
* react-mixin

## setup
```
npm install
```

## run the project
```
npm start
```

## insert keys from your parse.com account
```javascript
// Insert your app's keys here:
Parse.initialize('APPLICATION_ID', 'JAVASCRIPT_KEY');
```
